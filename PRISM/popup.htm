Pop-up Blockers

PRISM uses pop-up windows to display choices and options to the user. Because users are likely to have pop-up blockers installed on their computers, knowing how to over-ride these defaults will enhance the user’s experience within PRISM.

Pop-up blocking can occur on two different levels, within the browser application itself, such as Microsoft Internet Explorer (IE) and sometimes within plug-in software, such as the Yahoo and Google toolbars.

Due to the commonality of IE, it has become a target for hackers, requiring frequent automatic security updates. These automated updates, specifically in Microsoft Service Pack 2 (SP2), can possibly over-ride pop-up blocker settings previously chosen by a user. While the security updates protect users from general malicious attacks on the internet, they can disable standard functionality in web applications such as the CRL Image and Video Library.

For users with SP2, the default for the IE pop-up blocker is on with a Medium Security setting, meaning that it will block most automatic pop-ups. When activated, the pop-up blocker will play a sound and show the yellow Information Bar (just below the address bar). Some items that may trigger the blocker are: Installing an ActiveX control on the computer, Opening a pop-up window, and Downloading a file to the computer. This Information Bar provides details about downloads, blocked pop-up windows, potential security risks, and other activities, and allows a user to define whether to allow pop-ups or downloads from each site. Users can adjust these settings to reduce the blocking occurrences.

The browser plug-ins, such as the Yahoo and Google Toolbars, can be a hidden installation as part of another application installation, such as Yahoo Instant Messenger. The user may not even be aware of the existence of the plug-in on the computer. Becoming aware of common browser security trends will increase an administrator’s ability to troubleshoot user issues.

Adjusting Pop-up Blocker settings - Internet Explorer

1. Open Internet Explorer.

2. On the Tools menu, select Pop-up Blocker.

3. Click Pop-up Blocker Settings.

4. Click on Turn Off Pop-up Blocker.

Adjusting Pop-up Blocker settings - Netscape/Mozilla

1. Open Netscape/Mozilla browser.

2. On the Tools menu, point to Options.

3. Click Content.

4. Remove the check next to Block Popup Windows.
